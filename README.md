📚 Παιχνίδι Γραμματικής: Ουσιαστικά vs Επίθετα
Ένα διαδραστικό, εκπαιδευτικό παιχνίδι Drag and Drop σχεδιασμένο για μαθητές δημοτικού, με σκοπό την εμπέδωση της διαφοράς μεταξύ ουσιαστικών και επιθέτων.

🎮 Πώς παίζεται
Έναρξη: Πατήστε το κουμπί 🚀 ΞΕΚΙΝΑΜΕ! στην αρχική οθόνη.

Ταξινόμηση: Στο κέντρο εμφανίζεται μια λίστα με 20 τυχαίες λέξεις.

Drag & Drop:

Σύρετε τα Ουσιαστικά (πρόσωπα, ζώα, πράγματα) στο αριστερό πλαίσιο.

Σύρετε τα Επίθετα (ιδιότητες, χαρακτηριστικά) στο δεξί πλαίσιο.

Ανατροφοδότηση:

Αν η απάντηση είναι σωστή, η λέξη πρασινίζει και εμφανίζεται ένα 👍.

Αν είναι λανθασμένη, η λέξη κοκκινίζει και επιστρέφει στη λίστα σε τυχαία θέση.

Νίκη: Μόλις ταξινομηθούν όλες οι λέξεις, εμφανίζεται η οθόνη συγχαρητηρίων με εφέ κομφετί!

✨ Χαρακτηριστικά
Έξυπνη Δεξαμενή Λέξεων: Περιλαμβάνει πάνω από 100 διαφορετικές λέξεις.

Τυχαία επιλογή: Κάθε φορά που πατάτε "Παίξε Ξανά", το παιχνίδι επιλέγει μια νέα ομάδα λέξεων, ώστε να μην είναι ποτέ το ίδιο.

Visual Learning: Χρήση χρωμάτων και animations για άμεση επιβράβευση ή διόρθωση.

Responsive Design: Λειτουργεί άψογα σε υπολογιστές και tablet (διαδραστικούς πίνακες τάξης.
